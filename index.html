Portfolio






DirectMe


DirectMe is a web application that uses Yelp API, Weather API and Twilio API. This app randomly generates a place for someone to go based on his or her city, category, price range and distance. Special features that this app has are a randomize "DirectMe again" button (in case one is not satisfied with their first option), a text information to one's phone and a link to the company's Yelp page.
This application was built by four programmers and was created in less then a week. DirectMe took second in a week long hack-a-thon.
Technologies used: Python, CSS3, Materialize, JS, jQuery, MySQL, Yelp API, Twilio API, Weather API, Pylot MVC

Jumpster


Jumpster is an iOS application that is used to test the static vertical of an individual. By using Core Motion framework it allowed the app to measure airtime and vertical distance.
This app is a great social app, simply place phone on your stomach and jump! Special features include a toggle from centimeters to inches, a slider which customized the magnitude, and a reset button.
Jumpster was created by a group of four people. This app won 1st place in a 24 hour hack-a-thon.
Technologies used: Swift 3, Core Motion, Xcode 8

WeatherApp


Weather application was built using AngularJS 2.0. This application allows one to type in any city and it will return the temperature forecast for up to 7 days. This app will give the morning, afternoon, night times temperature, as well as, the minimum and maximum temperature of that day.
Technologies used: AngularJS 2.0, Weather API, Bootstrap, HTML, Javascript.
